# دليل إعداد APK والإشعارات

## خطوات رفع APK لقاعدة البيانات:

### 1. رفع APK إلى Supabase:
1. افتح `upload-apk.html` في المتصفح
2. اختر ملف APK الخاص بك
3. اضغط "رفع APK"
4. سيتم رفعه في مجلد `avatars` باسم `lv12apk`

### 2. إنشاء APK من مشروعك:
```bash
# استخدم build-app.bat
build-app.bat

# أو استخدم PWA Builder:
# اذهب لـ https://www.pwabuilder.com
# أدخل رابط موقعك
# حمل APK
```

### 3. تفعيل الإشعارات لجميع العمليات:

الإشعارات ستظهر تلقائياً عند:
- إضافة منتج جديد
- تحديث منتج
- طلب جديد
- تحديث حالة الطلب
- مستخدم جديد
- تحديث المحفظة
- رفع APK جديد

### 4. اختبار النظام:
1. افتح الموقع في متصفحين مختلفين
2. أضف منتج جديد من أحدهما
3. ستظهر إشعارات في الآخر فوراً

### 5. ملفات تم إنشاؤها:
- `upload-apk.html` - رفع APK
- `config.xml` - إعدادات Cordova
- `build-app.bat` - بناء التطبيق
- `manifest.json` - PWA
- `sw.js` - Service Worker

### 6. الاستخدام:
- المستخدمون يحملون APK من `download-app.html`
- الإشعارات تعمل تلقائياً
- التطبيق يعمل كـ PWA أيضاً
